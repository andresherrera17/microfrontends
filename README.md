# microfrontends

1. npm install --g create-single-spa
   - crear nuevos microfrontens
   - configurarlos

2. ejecutar en todos los microfrontens npm install
   
3. ejecutar cada microfrontend en su puerto
   - header -> 4200
   - main -> 4202
   - screens -> 4203

4. Ejecutar container -> npm start
